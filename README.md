# Profile Screen App - Week 2

A simple Android application built using Kotlin that allows users to view and edit profile details such as name, email, and bio. The app demonstrates basic Android concepts like activities, navigation, form validation, and local data storage.

#### To change profile picture navigate to app/src/main/res/drawable
